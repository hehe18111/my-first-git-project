# Fullstack Todo App

## Tech Stack
- **Frontend**: React (Deploy lên Vercel)
- **Backend**: Node.js + Express (Deploy lên Render)
- **Database**: MongoDB Atlas (Cloud)

## Features
- ✅ User authentication
- ✅ CRUD todo operations
- ✅ Real-time updates
- ✅ Responsive design

## Deploy Strategy
- Frontend → Vercel (tự động từ GitHub)
- Backend → Render (tự động từ GitHub)
- Database → MongoDB Atlas (cloud)
